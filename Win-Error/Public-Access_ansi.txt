

Чтобы включить обнаружение компьютера в сети, 
установите для необходимых служб тип запуска Автоматически.

Сервер
Хост поставщика службы обнаружения
Публикация ресурсов обнаружения функции
Обнаружение SSDP
Узел универсальных PNP-устройств
Модуль поддержки NetBIOS через TCP/IP



Панель управления > Сеть и Интернет > Центр управления сетями и общим доступом
> Изменить параметры адаптера > Свойства > 
	Internet Protocol Version 4 (TCP/IPv4) на вкладке Сеть
Выберите Дополнительно, выберите Включить NetBIOS через TCP/IP на вкладке WINS.


Панель управления > Администрирование > Службы компонентов
: Службы компонентов > Компьютеры > ПКМ > Свойства
: Свойства по умолчанию
:: + Разрешить использование DCOM на этом компьютере
:: - Включить на компьютере Интернет-службы COM
:: Уровень проверки подлинности по умолчанию > По умолчанию
:: Уровень олицетворения по умолчанию > Олицетворение

Панель управления > Локальная политика безопасности
: Локальные политики > Параметры безопасности
:: Сетевой доступ: разрешать применение разрешений 
	"Для всех" к анонимным пользователям > Включен
: Локальные политики > Назначение прав пользователя
:: Отказать в доступе к этому компьютеру в сети > удалить все ПК


Касперский > Защита > Сетевой экран > Сети
: Сетевой адаптер > Доверенная сеть
