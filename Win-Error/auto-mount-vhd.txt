Вот готовый скрипт:
@cd/d "%~dp0"
@echo off

set VHD_PATH=C:\keys.vhd
set DRIVE_LETTER=F

:: 1. Подключаем VHD
(echo select vdisk file="%VHD_PATH%"
echo attach vdisk) | diskpart > nul

:: 2. Получаем номер диска, который появился (последний виртуальный диск)
set "SCRIPT=%temp%\vhd_get.dp"
(
echo select vdisk file="%VHD_PATH%"
echo detail vdisk
) > "%SCRIPT%"

for /f "tokens=3" %%i in ('diskpart /s "%SCRIPT%" ^| find /i "Disk ID"') do set DISK_NUMBER=%%i

:: 3. Принудительно назначаем букву F
(
echo select disk %DISK_NUMBER%
echo select partition 1
:: Если буква уже занята другим томом - сначала снимем её
echo remove letter=%DRIVE_LETTER%
echo assign letter=%DRIVE_LETTER%
) > "%SCRIPT%"

diskpart /s "%SCRIPT%" > nul

del "%SCRIPT%" 2>nul

echo VHD подключен, диск %DISK_NUMBER%, буква %DRIVE_LETTER% назначена.
Пояснения:
• remove letter=%DRIVE_LETTER% – если буква F была занята другим томом, команда её освободит (игнорируя ошибку, если не была занята).
• assign letter=%DRIVE_LETTER% – присваивает букву F вашему VHD.
• Вся работа выполняется через diskpart с правильной передачей команд.
Сохраните как mount_vhd.bat, запускайте от имени администратора. При каждом запуске VHD будет подключаться и буква F будет принудительно присвоена ему (если на момент подключения буква была занята – освободится и достанется VHD).
Если хотите, чтобы сначала отключался уже подключенный VHD (чтобы избежать ошибки), добавьте в начало:

(echo select vdisk file="%VHD_PATH%"
echo detach vdisk) | diskpart > nul 2>&1

в Windows 7 нет встроенной галочки в интерфейсе, чтобы просто добавить программу в автозагрузку с правами администратора. Однако эту задачу можно решить, и есть два основных способа: через Планировщик заданий и через оснастку «Автозагрузка».
Сразу отмечу, что оба способа потребуют однократного ручного вмешательства. После этого скрипт будет запускаться автоматически каждый раз при включении компьютера.

Способ 1: Планировщик заданий (Самый надёжный)
Этот метод считается самым правильным, так как Планировщик заданий был создан Microsoft именно для таких задач.
1. Откройте Планировщик заданий: Нажмите Пуск, в строке поиска введите taskschd.msc и нажмите Enter. Система может запросить подтверждение прав администратора — согласитесь.
2. Создайте задачу: В правой колонке (панель «Действия») выберите «Создать задачу...».
3. Настройте общие параметры (вкладка «Общие»):
• В поле «Имя»: дайте понятное имя, например, MountVHD.
• Обязательно в нижней части окна установите флажок «Выполнять с наивысшими правами»-. Без этого скрипт не получит прав администратора.
4. Настройте триггер (вкладка «Триггеры»):
• Нажмите кнопку «Создать...».
• В выпадающем списке «Начать задачу» выберите «При входе в систему».
• Убедитесь, что в разделе «Дополнительные параметры» выбран «Любой пользователь».
• Нажмите «ОК».
5. Настройте действие (вкладка «Действия»):
• Нажмите кнопку «Создать...».
• В поле «Программа или сценарий» укажите путь C:\Windows\System32\cmd.exe.
• В поле «Добавить аргументы (необязательно)» вставьте следующую строку:
/c "путь_к_вашему_файлу\mount_vhd.bat"
(Обязательно укажите реальный путь к вашему BAT-файлу и оставьте кавычки, если в пути есть пробелы).
• Нажмите «ОК».
6. Завершите создание: Нажмите «ОК» в главном окне создания задачи. При необходимости система может запросить пароль вашей учётной записи — введите его, чтобы задача могла выполняться в фоновом режиме.
После этих действий ваш VHD-диск должен подключаться автоматически при каждом входе в систему.

Способ 2: Ярлык в автозагрузке
Этот метод проще, но менее элегантен: при загрузке системы на долю секунды может появиться окно командной строки.
1. Создайте ярлык для вашего BAT-файла: На рабочем столе или в любой папке нажмите правой кнопкой мыши → «Создать» → «Ярлык». Укажите путь к вашему mount_vhd.bat.
2. Настройте ярлык на запуск от имени администратора:
• Кликните по созданному ярлыку правой кнопкой мыши и выберите «Свойства».
• Перейдите на вкладку «Ярлык».
• Нажмите кнопку «Дополнительно...».
• В открывшемся окне установите флажок «Запускать от имени администратора»-.
• Нажмите «ОК» в обоих окнах для сохранения настроек.
3. Поместите ярлык в автозагрузку:
• Нажмите клавиши Win + R, введите команду shell:startup и нажмите Enter. Откроется папка автозагрузки вашего пользователя.
• Скопируйте или переместите в эту папку созданный ранее настроенный ярлык.
Важно помнить: при использовании этого метода во время запуска системы может появиться стандартный запрос UAC (Control User Account) с просьбой подтвердить запуск от имени администратора. Если он появляется, нажмите «Да».
